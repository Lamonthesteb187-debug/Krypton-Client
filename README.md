# 🌌 Krypton Client

An advanced utility and automation client designed for modern Minecraft Java Edition (1.21+). Krypton isn't just another generic PvP client—it is built from the ground up for SMP, Factions, and Economy-focused servers, offering specialized automation tools to handle the heavy lifting of base hunting and economy management.

> **⚠️ Disclaimer:** This project is intended for educational, research, and private server testing purposes only. Use on public competitive servers may violate their terms of service and result in account bans.

---

## ✨ Key Features

### 🔍 Elite Base Hunting & Stash Finding
* **Smart RTP Scanner:** Pairs directly with server `/rtp` commands to instantly evaluate newly loaded chunks for player-placed containers, chests, and infrastructure.
* **Seed-Assisted Analyzer:** Evaluates natural world generation data to instantly flag chunk anomalies, hidden underground trails, and player activity.
* **Filtered ESP Overlays:** High-contrast, performance-optimized visual overlays that cut through the noise. Target specific high-value blocks like Ancient Debris, Mob Spawners, and hidden chests.

### 💰 Economy & Market Automation
* **Auction House Sniper:** An ultra-fast market scanner that monitors `/ah` listings and automatically snipes mispriced items, rare gear, and bundles.
* **Inventory & Automation Macros:** Built-in scripts designed to optimize chest sorting, fast-selling, and material processing to dominate server economies with zero downtime.

### ⚙️ Stealth Network Management
* **Modular Click-GUI:** A clean, lightweight layout that lets you tweak configurations, filter search terms, and toggle modules on the fly.
* **Configurable Packet Smoothing:** Highly adjustable movement, inventory, and action modules built to communicate cleanly with modern network architectures.

## 🛠️ Installation & Quick Start

1. **Prerequisites:** Ensure you have a standard Minecraft Java environment set up.
2. **Download:** Grab the latest official build from our Releases tab.
3. **Launch:** Place the client files into your `\.minecraft\mods` folder (or inject via your preferred launcher).
4. **Open Menu:** Once in-game, press `Right Shift` (default bind) to open the Click-GUI and begin configuring your modules.
